还算充实的第一周

发表于 2017-03-08 | 分类于 生活 |

3月8日凌晨00：46，今天是妇女节，等会写完博客得早点睡了，因为明天早上要争取早点起来，给妈妈说一句节日快乐，让她晚上和爸爸出去搓一顿，顺便和老爸看个电影，没办法，因为身为儿子，没办法在她身边陪她过节。

离职一周的状态

离职刚刚满一周，说说这一周的状态吧。回来的那天因为是周五，连着周末两天，我把南京的出租屋好好打扫了下，也把这个所谓的独处空间布置了一番，毕竟之后不知道得在这屋子里过上多少天无业青年的日子。

突然间时间多了起来，反而会让自己有些措手不及，我开始进行一系列的规划和计划，比如之前的博客，就是我这些规划和计划的成果，但计划归计划，执行力是另一回事了。就像stormzhang说的，如何做到脱颖而出的五条建议，最后一条就是执行力，毕竟“道理我都懂，可是我做不到”。诚然，我对我自己还是了解的，打败我的无非是几个念头：打几盘DOTA休息一蛤不过分；学习了几小时了太累听会歌；MB的一个人在房间实在他妈的太无聊寂寞了，找几个兄弟吹个牛逼啊…

但每次将这些打败我的念头执行完后，我的内心又会被自责所占据，细思之后，我对我现阶段的这个状态给出了定论：我是没办法克制自己的。给出这个结论实际上并不容易，因为这是一种自我否定，而且还得摒弃那些自我陶醉的尊严，然后啪的一下给自己三耳光子：“小之你特么真是个废柴”！

一次不妥协的离职决定

发表于 2017-02-28 | 分类于 生活 |

以下内容的所有公司名、项目名和银行名都不具体给出了，保密协议签了

我刚从公司办完离职手续回来，走在路上，想了很多。

本来我是不打算离职的，但我的人生，存在着不妥协和不将就

作为一个在大学为了一份看似飘忽的梦想—-“电子竞技”而奋斗了四年的毕业生，我的技术水平实际上是很烂的。可想而知，我在大学除了大一在什么高数上上过几节课以外，其他课基本上没有上过，甚至毫不夸张的说，我甚至想过辍学去打职业，为了自己这份执着和努力，也可能是为了向身边的人证明什么：我所做的事不是你们想的那样，只要努力，任何一行都可以让你们刮目相看

当然，就如大家想象的那样，我妥协了，妥协于父母，妥协于身边的目光和评价，妥协于这个社会…我放弃了这样的机会，后来每年看到新闻上各种有关为电子竞技正名的报道，我只是笑笑，一切都无法重来，即使所有人都认可了，但我的时间，我的努力，都已经成了过去，还有我的青春，我痛恨自己的妥协。有关我这一段宝贵、可爱，充满热情的这段电子竞技经历，这些都是后话，以后有时间我会单独拿出来写的，这里就先不多说了。

回归正题，今天刚刚办完离职手续，实际上对于这份工作，虽然只有短短的8个月，但我还是有很多感悟的，毕竟是我的第一份工作，也算是人生的一个全新阶段的实践。

公司的性质是外包软件公司，给银行做外包，比如制作手机银行、网上银行，直销银行这样的软件。我在公司担任的是安卓开发工程师，负责开发上面这些软件的安卓客户端。说来也巧，实际上，当时在校招的时候，我的技术基本上是一个完全菜鸟，在大学时候的一些项目基本上我只做过一些需求文档和数据库的工作，因为我一般都和胜哥（我的大学室友，现在是华中科技大学博士，真正的牛中牛人，非常努力，非常厉害）一组，抱着他的粗腿，大学也就这么蹭过来了，所以逻辑代码的开发经历基本为0，但就在这种前提下，凭借着我的三寸不烂之舌，我还是通过了公司的面试，去年毕业季的3月份就拿到了工作，不得不说很幸运，至于这应聘内容您也就别问了，从头到尾我都没有被问到过技术问题，我的确试图把面试官的话题往别的地方偏，比如说我考过研究生，专业课多好多好云云，多想去南京云云，总之乱吹一通，就过了。后来为了做毕业设计，我自学了安卓，并独立开发了一个理财软件，通过了毕设，然后7月份入职培训，几次考试综合排名第一，在所有的新员工中，我是第一个被派送到项目组的。

非静态内部类为什么不能有静态成员

发表于 2017-01-13 | 分类于 技术 |

看《Thinking in Java》的时候，根据书上代码进行编码的时候，发现了一个报错。

在A类这个外部类里，非静态内部类拥有一个静态的整型成员变量i，这里编译不通过，编译器报：

inner classes cannot have static declarations

解释过来就是，内部类不能拥有静态声明。但我们知道，静态内部类是没有这方面的限制的，不仅可以拥有静态声明，也可以拥有非静态声明，那么Java编译器为什么不允许非静态内部类拥有静态声明呢？

2017元旦-我的上海首拍之行(第二天)

发表于 2017-01-07 | 分类于 生活 |

好吧，因为这几天一直在忙别的事，所以晚了好几天才有时间来发上海之行第二天的博客。实际上第二天的经历要比第一天要更加丰富一点，因为第一天是12月31号跨年夜，人非常多，想拍一个取景很满意的照片非常难。而且人多起来，旅行的心情也会大打折扣，不断的在人群中穿梭不仅累，出来放松的初衷也违背了。

在地铁站门口的港式咖啡小店买了喝咖啡，不得不说，记忆中的香港和上海的气质有点像哦，这家上海的香港咖啡厅完美的融合了两者的特点，小店风格很有意思，注意到还有大话西游的相片呢！
