还算充实的第一周

发表于 2017-03-08 | 分类于 生活 | | 本文浏览量 次

3月8日凌晨00：46，今天是妇女节，等会写完博客得早点睡了，因为明天早上要争取早点起来，给妈妈说一句节日快乐，让她晚上和爸爸出去搓一顿，顺便和老爸看个电影，没办法，因为身为儿子，没办法在她身边陪她过节。

离职一周的状态

离职刚刚满一周，说说这一周的状态吧。回来的那天因为是周五，连着周末两天，我把南京的出租屋好好打扫了下，也把这个所谓的独处空间布置了一番，毕竟之后不知道得在这屋子里过上多少天无业青年的日子。

突然间时间多了起来，反而会让自己有些措手不及，我开始进行一系列的规划和计划，比如之前的博客，就是我这些规划和计划的成果，但计划归计划，执行力是另一回事了。就像stormzhang说的，如何做到脱颖而出的五条建议，最后一条就是执行力，毕竟“道理我都懂，可是我做不到”。诚然，我对我自己还是了解的，打败我的无非是几个念头：打几盘DOTA休息一蛤不过分；学习了几小时了太累听会歌；MB的一个人在房间实在他妈的太无聊寂寞了，找几个兄弟吹个牛逼啊…

但每次将这些打败我的念头执行完后，我的内心又会被自责所占据，细思之后，我对我现阶段的这个状态给出了定论：我是没办法克制自己的。给出这个结论实际上并不容易，因为这是一种自我否定，而且还得摒弃那些自我陶醉的尊严，然后啪的一下给自己三耳光子：“小之你特么真是个废柴”！

好吧，所以认清自己是啥货色也不是啥坏事，至少解决方法也粗暴了很多，也就别走啥文艺或者是鸡汤型克制了，粗暴解决一些根源性欲望源泉最为直接，例如：把DOTA2删了，然后在几个玩游戏的基佬厮混的微信群里抛出一句：“在下要去复习找工作了，近日先鸽一记”，然后饱含深情和留恋的退了群，给自己发了个鸡汤，敲特么的代码去吧。

这个月的大方向

这某同事在和我聊天的时候可是给我灌输了，3月份是找工作的黄金时间，招人多，工资高。虽然我是可以理解这3月份招人因为毕竟是春招黄金期，人肯定是会多的，但这工资高了我却不是很理解，主要在咱这IT界，你工资高低，还是和你这技术能力和市场环境这两个因素分不开的，高能高哪去呢？不过总之，他这一说，我得到的信息就是，现在是个找工作的好时间，过了这个村，可能就搭不上好车了这样感觉有点误导性质的信息，虽说我心里觉得是误导，但潜意识却不争气的有了压力，所以为了心安，不让自己又瞎B想那些还没发生但可能发生的事（这是我一个大毛病，心里留不住事），我就又对我这找工作大计立了一些FLAG，譬如：3月中旬开始找工作；只找产品公司这样的。

我这样列计划也是有一定依据的，首先，外包是个坑这点是扎根在我心里的，总之，互联网公司 > 国企 > 外包，产品公司至少还是比较稳定的，而且如果蹭上了个上升期公司，里头的潜力不用多说。再有就是，这一周虽说我的复习专注度不是很强，但复习效果却还不错，自己重新搭建了以前项目上的框架，思路上过了一遍，也把很多重点和知识盲区复习到了（有一句说一句，这也有可能是我专注度分散的所给自己找的借口和台阶，自己说服自己总有一万种理由），所以感觉可以在这个月内差不多就可以找工作了。

健身

回来南京之后，有一件事重新回到了正轨，那就是健身。隔一天去一次，流一次汗，洗一次热水澡，这可能是这单调寂寞生活中的少有的多色彩的行为了。我的健身项目很简单，2-3公里的跑步，随意的做一些器材，做一些简单的拉伸，拉一拉韧带，流了汗，酸了手，就够了。保持好运动量，维持自己的健康是初衷，我不追求体形上的塑造，因为我现在的体形还算是让人看起来顺眼的。

运动几次后觉得自己的上肢力量还是比较弱，导致做一些胸腹和背部肌肉训练的器械和动作都会很吃力。同时，我的韧带实在太硬，这是从小到大的问题，韧带的拉伸对人体健康是非常重要的，所谓“筋长一寸，寿长十年”，忍着痛压压，还是很必要的。最后，我的颈椎和腰板都是有一些偏曲的，这和我平时久坐以及一些不好的坐姿站姿习惯不无关系，这是个长久难题，需要长期持之以恒的进行矫正，同时这也是个痛苦的过程。

小密圈的加入

我花了129元加入了stormzhang的小密圈，总之，平时我看帅比张的文章最多，也的的确确的受到了他很多启发，我觉得他很亲切，因为他和我们一样，都是从最平凡的程序员做起，慢慢的通过自己的努力，达到了一些不错的成就，甚至他的起步更低，他的文章和博客大多都是已自己切身经历说起，非常贴近我的日常生活和工作，很有借鉴意义，我想，这也是大部分关注他的粉丝选择关注他的原因。

当然，既然这个人给了我帮助，对我有意义，那么，花一些钱订阅他的小密圈对我来说一定是值得的，而且我觉得日后的收益一定远远大于这129元，同时，这样做的行为也算是一种自我投资，stormzhang也提到过自我投资这种话题，总之，自我投资是值得的，是可以花任何代价去实践的。这世上的确有很多便宜甚至是免费就能获取到的东西，不排除这里有很多干货，比如现在信息时代的碎片化知识，但能花钱买到的，比如订阅一些作者的文章，或者动辄千元的教学视频课程；花贵的钱买到的，比如宝马和奥拓，iPhone和小米；这些东西需要钱、比别的东西贵，一定有他的道理，所谓的性价比都是没钱的人在意的，如果我很有钱，我穿衣服会穿那些淘宝1，200买就能买到的所谓的性价比款？？我想应该不会，那些数千元的衣服一定有它不一样的材质和设计，我有钱，我就有权利选择更优质、更有品质的生活，而且，为什么不去这样做呢？

当然，我的想法是跟上了，我的钱包却还暂时落伍着。不过，在力所能及的范围内，我会选择尽量让自己接触优质的资源，比如这次小密圈。

与我在意的人交流

先说点题外话

我这个人现在越来越极端化，好听点说是一种追求极简生活理念的状态。比如最近在用微信的时候，就萌生出精炼我的朋友圈的想法，将一些联系少的；没啥好聊的；性格不合的；鄙视的；蔑视的；无视的；不想理的；厌烦的那些“好友”全部一清而空，做到所谓的，只和我认可的；我在意的；思想层次水平相近的；关系好的留下来，所谓留下精华，省事省心省力。而且我觉得这也是一种相互的过程，如果别人觉得我是个垃圾，那我要知道了，我只会劝他赶紧清空我，咱为他节省了时间和精力，免得回头人还嫌我在他朋友圈里碍着，一举两得，皆大欢喜，互惠互利，简单粗暴，共产主义，何乐不为。

不过我也认识到了我现在性格的极端化，我意识到了，所以才将他陈述出来，也是一种自省。

大男子主义

这周有几次走心的交流有些感触，比如一位女性朋友跟我说了真心话，她认为我有一些大男子主义，具体现象就是：过于自主自立理性主见，虽然对女生很温柔很会关心对方，但对女生有一种控制欲，极大的降低了一个女生在我身边的存在感，她的原话是：你会弱化女性作为一个单独的、自主的，与男性同样的独立人格的存在，你会让女生的存在感变弱。总之，我这样的性格不好办，她说女生太自我难以被我掌控，毫无主见的女生也会让我缺乏兴趣。老实说，在她跟我说这些话之前，我甚至都不知道大男子主义所代表的性格是什么意思，我一切的行为都是我主观个性使然，可能就像她说的那样：可能我已经意识到了我性格上的一些缺点，但不知道如何是好，所以直接接受了吧。

她的话调动了我很多记忆，很多有关我性格上的记忆，但到底该怎么面对这个问题，如何理解这样性格的我，我现在仍然在思考，我甚至不完全了解我自己。

霸道的人生态度，并不是一种冲动

就在几小时前，另一个朋友向我哭诉，她刚刚换工作两个月，但新的这份工作每天都加班到10点，而且还住着集体宿舍，让她非常不适应，所以她想问我的意见，到底应不应该辞职。她用富含一堆QQ表情的言语（大多数是哭表情和难过表情）描述的工作场景充斥着她各种抱怨和不满。她还说，所有朋友都劝她再多忍耐忍耐，千万别冲动。只有她爸爸心疼她让她尽快辞职。

如果这么不满意，为什么不辞职呢？觉得不舒服，不满意，很难受，然后辞职了，去追求适合自己的，难道不是很正常的思路和途径吗，为什么是一种冲动？难道你喜欢畏首畏脚的做决定？

难道中国青年这么愿意妥协于企业（或者说社会），甚至甘愿牺牲自己的一切期望？你们作为当代青年，就这么没有尊严和底气吗？难道辞了职，就没办法找到你用武之处了吗？你就这么看扁自己？

我想了一下，跟她说：如果不接受，就辞吧，没什么大不了的。

她感激涕零的跟我说：“好感动你这么支持我，我就怕那些让我不要离职的劝阻。”

她问我的时候，对于我即将作出的回答，她可能心中已经有了一个期待的答案，那个答案就是她的选择，实际上她早已经选择了。

我想，可能大多数人在做一些非常困难的选择的时候，可能心中已经有一个答案了。但需要一个契机来坚定它。
