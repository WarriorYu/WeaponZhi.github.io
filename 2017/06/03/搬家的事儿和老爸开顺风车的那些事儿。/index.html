搬家的事儿和老爸开顺风车的那些事儿。

发表于 2017-06-03 | | 本文浏览量 次

老爸和我总算搬完家了，端午节期间老妈也过来南京和我一起过了。我们一家一起去宜家买了一些家具，也把之前在南方花园的一些家具搬了过来，后来我又到淘宝上买了壁画和墙贴，有天晚上下班后叫上同事来我家帮我弄了一下，最后挺满意的，很温馨，也很有氛围。

房子的装修风格实际上参考了知乎的一个回答，那个问题是「如何花最少的钱提升出租屋的格调」。

如何花最少的钱提升出租屋的格调–王二里的回答

老实说我从南方花园到搬到这里，我很多布置都是借鉴了这个回答，也做了一定的修改，最大的不同就是我的工作台，因为我是 IT 工作者，所以我的工作台当然是科技感十足，全是最好最棒的数码产品。放几个图吧。


新家的布置我还有很多话说，因为明天要调班，所以准备等下周的时候在知乎那个问题下写一篇自己针对这个问题的答案。

老爸前几天从连云港过来，很迟才到，平时他都是4,5点就到南京了，那天晚上 10 点多才到，到了家，跟我坐在餐桌前吃饭，他说今天遇到了很多让他很感慨的事情。

第一件事：早上老爸接的第一个客人，是他在前一天晚上已经约好的。一大早老爸就开过去接她了，大概有4,50分钟的路程，一路上都和这位客人保持畅通，等老爸到了离她只有 100 米左右距离的时候，她却把订单取消了，再也联系不上了。老爸说这是一种恶意取消单子的行为，她可能是故意的。

毫无预兆。

第二件事：老爸那天一共接了四个客人，哪个时间去接谁，我爸规划的很明确，现在少了一个人，如果最后只把这 3 个人接到南京，老爸可能不咋赚钱，但如果再去接一个新客人，可能时间就不够了。但老爸最后还是接了一个县城里的一个客人，来回可能要1个多小时，还好另外三个客人有两个是一起的，和他们商量了后，他们同意稍微迟一点。

协商后，老爸就快马加鞭去接那个新客人，然后再去接那三个人。这三个人里面有个女孩，要到南京禄口机场坐飞机，虽然老爸去南京路上已经一直在提速了，甚至有些路段都飙到 140 了（后来我给我爸说，以后不管遇到啥情况还是别这么急，安全永远第一），但很可惜的是中间遇到一段路堵车了几十分钟。到机场后，差了十几分钟，不过女孩在到南京之前已经改签好换了一班飞机了。

从始至终，她一句话都没有抱怨过，还不断的跟我爸爸说，不要急，慢点开，没事的。她说实际上上车的时候就感觉可能来不及了。

老爸说特别佩服这个女孩，很感激，也很感动。

两件事，两个人，截然不同，在同一天。
