来南京一年，我就要搬家了

发表于 2017-05-25 | | 本文浏览量 次

一

来南京快一年，我在我那可爱的出租屋里，也住了快一年了。这一年是我人生中进入社会的第一年，发生了很多事，不过就说说这房子吧，虽然不大，但很温馨，我也花了很多时间去有意识的布置它。

虽然我是和一个有着 6，7 年关系的哥们一起整租的这个房子，但对我来说，只要我进入了自己的房间，这个房间就是一份独处的空间。所以我会去逛逛宜家，买一些家居，买一些漂亮的装饰物打点我那间十几平米的独处之所。

我是一个充满孤独感的人，但却是一个需要独处的人。

虽然我比较开朗，也能和很多人交朋友，在朋友眼中我也是个能侃侃而谈的人，有时候也可以和人称兄道弟，满嘴跑跑火车。但私下里，我有自己的个人世界，我的内心世界是神圣的，不容外人踏足。所以，我需要独处。

但习惯独处的人，就要习惯寂寞和孤独。进入了那扇门，就进入了另一个世界，在这个世界里，你是主宰，所以，没有人能限制你的任何欲望，如果你不能自律，你就会变得被欲望和精神支配。

不得不说，我一开始的确沉沦于这个世界中，我可能不分昼夜的在玩游戏，或者昼夜颠倒，白天睡觉，晚上做事。比较特殊的是，我的这个房间是不透风不向阳的隔间，当初经验少，阴差阳错就选了这个房间，所以本身在这个房间里如果门窗紧闭，是根本无法分辨外面是白天还是黑夜的。

刚刚看到一句话说的挺好，情商高的人，思维密度都很高，做事情又快又好，经常给人惊喜。这世界上大多数的蠢人，一般都是懒人。

一个自律的独处者，从内到外，你都很难击溃他。

如果不自律，那就是在自掘坟墓，自我认可的失败者。

但有时候我的状态又非常适合在这里，因为这种环境适合自己思路打开，去想象，去探索，去寻找自己心中的那个世界。

我们身处的这个世界，往往不是我们所认识的那样，我想去尽力的去看看这个世界到底是什么样的，如果把这个世界完全的客观化，我们每个人的存在不依附于任何人，那么这是多么的纯净。人类太把自己当回事了，把自己跟世界完全脱离了，去做世界的支配者和探索者，实际上我们本就是这个世界的一部分，每一个个体的状态既是这个世界的某一种状态的永恒，为什么不试图去珍惜呢。

所以说，人类能改受到时间的流逝真是莫大的嘲讽。

二

越写越偏题。拉回来吧。

爸爸现在是一个顺风车司机，每隔一天都会来一次南京，但和我住一个房间确实有诸多不便。
一来，他知道我比较习惯独处。我也比较大了，父母离我太近也不是特别好。
二来，老爸作息时间和习惯和我不太相符，所以也不适合住在一个房间。

所以上周末我和爸爸就找一天去找了下房子。最终选定了一个房子，价格很便宜，60平的两室一厅，1600一个月。要知道，我现在住的两室一厅，格局差，平方小，最后2400一个月。不过便宜也有它的道理，主要就是这个小区没有地铁，上班的话只能骑电动车。虽然我的捷安特单车被狗日的小偷偷了，但我爸爸也把我的电动车从连云港运了过来，昨天擦了擦，换了个车篮，测试了下25分钟左右到公司，还是可以接受的。

这个新房房东非常用心，改造了一些格局，让空间变得更大了，同时装修风格也是非常用心，很多人性化的设计让人觉得特别舒服，比如插座的位置，开关的位置，还有一些置物的小隔层的设计，都是让人非常满意。而且这个房子的格局非常规整，非常适合日后的一些小装修的微调。感觉自己大展身手的时候又到了。

当然最重要的，两个房间都是通风向阳的，厨房空间也适中，通风也很好，对于健康的生活方式，我们很认真。

三

忙了一天，老爸开车带我回家，路上跟我聊了很多，他说：

儿子，我和你妈这辈子实在太过平庸了，所以，如果你想去做什么，梦想去做什么，尽管放手去做，我们不会干涉你的，要去体验追求理想的过程。

老爸，你话说的是不错，但这是你的心里话吗？

大学那几年，每当我跟你们说，我真的很热爱电竞，我的梦想就是站在世界的巅峰，为电子竞技做出贡献。你们的态度往往是一阵冷嘲热讽。最后也因为你们的强硬反对，我放弃了这个热爱多年的事业。

我不怪你们，当时绝大多数的人都觉得我是个玩物丧志的败家子，何况是父母呢。但后来几年，随着电竞曝光度越来越高，奖金越来越高，正规化越来越高，你们也渐渐改变了态度，甚至后来还给我说过后悔当时这样阻拦我，应该放开手让我去做喜欢的事。

听到你们这样说我很高兴，但又有什么用呢，现在的我回得去吗？

父母永远是对的，父母的初衷永远是好的，但父母往往在这种事上只会求稳，而不会去也不愿意让你尝试一些有风险的事物。

中国的父母大多是这样的，但这样的结果就是，很多中国的孩子都是这样被爱扼杀的。

直到现在，电子竞技依然是我心中深藏的一股热流。我依然关注着这个圈子。

因为曾经爱过，曾经拼命过，但这份心结却因为父母的抵制而从未打开过。

如果他们当时愿意让我去试一试，也许，即使失败了，我这份心结也会已经打开了吧，这几年我可能也不会每次见到电竞方面的消息心情都会低落吧。

不过这也不能完全怪父母，怪这个中国社会，最应该责备的，就是我自己，我恨当时毫无魄力的自己。

四

不管怎么样，马上又是新的开始。人也是动物，居住所变了，顺其自然的就感觉生活即将产生变化。

我相信人生不仅有这些负能量，还会有很多值得我去发现的美好。

我也一直鼓励我自己，这个世界是平衡的，也许， 另一个平行世界的我正在电竞赛事上努力拼搏。

那这个世界的我，是不是也该努力的做些什么。
