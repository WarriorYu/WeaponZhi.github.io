2017元旦-我的上海首拍之行(第二天)

发表于 2017-01-07 | 分类于 生活 | | 本文浏览量 次

好吧，因为这几天一直在忙别的事，所以晚了好几天才有时间来发上海之行第二天的博客。实际上第二天的经历要比第一天要更加丰富一点，因为第一天是12月31号跨年夜，人非常多，想拍一个取景很满意的照片非常难。而且人多起来，旅行的心情也会大打折扣，不断的在人群中穿梭不仅累，出来放松的初衷也违背了。

但第二天就好多了

虽然是第二天，但实际上这一天是元旦，真正意义上的2017第一天，同时，身边陪我逛吃的妹纸也换成了大学同学。

第一站，衡山路

衡山路出名在酒吧街，还有大量高大粗壮的法式梧桐树，但可惜的是，我们从衡山路地铁站下站开始就走错了方向，所以酒吧没看到几个，随手拍了一些街景而已。

在地铁站门口的港式咖啡小店买了喝咖啡，不得不说，记忆中的香港和上海的气质有点像哦，这家上海的香港咖啡厅完美的融合了两者的特点，小店风格很有意思，注意到还有大话西游的相片呢！



衡山路近年来酒吧街荒废了很多，但路边曾经的建筑却在诉说着这条街曾经的繁华和喧嚣，就好像被人无意间随手遗弃的艺术品一般，虽然放在角落，但拿起来的时候，还是能品味出它曾经的美好。


暴露在墙上的老式信号箱

冷清的酒吧，光秃秃的树上零星的叶子，还有停在树下的蓝色版mobike

梧桐大道

外面是繁华的高楼大厦，转折处却嗅出了小时候的味道

时尚的小哥在等红灯，和满眼的梧桐

好吧，妹纸说累了，咱喝个奶茶吃个包子准备下一站吧！话说现在的奶茶小哥都帅的不行啊

第二站，田子坊

如果你去上海，田子坊是不得不去之处，田子坊是里弄民居味道，弄堂里除了创意店铺和画廊、摄影展，最多的就是各种各样的咖啡馆，不过因为今天是元旦节，所以逛吃田子坊的过程中还是比较痛苦的。路人太容易入镜了，想单纯的拍摄一张里弄味道的街景实属不易。

就好像90年代的家，但它却充满了商业气息

为什么人这么多啊！ T-T

各种创意小店和咖啡厅小酒吧

这张我挺喜欢的

我想几十年前，这里应该只有生活，没有游客

第三站，新天地

漫步新天地，仿佛时光倒流，有如置身于二十世纪二、三十年代的上海，但一步跨进每个建筑内部，则非常现代和时尚，亲身体会新天地独特的理念，这有机的组合与错落有致地巧妙安排形成了一首上海昨天、明天、今天的交响乐，让海内外游客品味独特的文化。 —引用百度百科

新天地的建筑都是根据曾经的石库门改造的，风味十足，却时尚袭人。这里还有中共一大旧址，年轻人，外国人都汇聚在这，这里是上海时尚的象征。

这么文艺的哈根达斯店面你见过吗

拐弯处是否还能遇到儿时的玩伴，想起你们曾经在楼与楼之间的小道间奔跑的场景？

无言

我把镜头对准他们很久，他们后来看到我的镜头后，笑了笑，接了吻，但我没有记录下来

如果有莫扎特和咖啡，我能在这个书店坐上一天也无妨。

厚重与张扬的冲突

潮流的集聚地

第四站，K11

这里是购物者的天堂，有着最高档的店面，同时，K11的建筑风格本身也是一件艺术品，K11里不仅有各种商店，还会展示各种各样艺术家的作品，以及很多风格别致的商品可以选择，让人大开眼界。

当然，有钱人也很多！

因为镜头的广角实在不够，所以拍出的气场不够强

极具别致的奢华餐饮店

各种玩物

离别之前，上海之夜

离开上海之前，趁单反还有电，赶紧最后留下了几张上海夜晚的记忆
石库门边

上海的夜晚提醒着你，新年来了，你应该开心点，迎接美好，看到身边的缤纷绚丽

同时还有前几天的圣诞节

k11流光奢华

上海的夜晚比白天热闹，灯火通明，昼夜无眠

总结

作为第一次用单反，第一次街拍，我觉得这次的上海之行真的已经很痛快了，当然，我也确定了摄影即将是我下一个爱好，写这篇博客的时候，我已经入手了第一个镜头，是用来拍人像的，我正在逐渐感触摄影的魅力，并正在通过摄影扩展我的世界，丰富我的内心世界，记录生活中的美好，本身就是一个让自己更美好的事情！


那么，上海再见，下次再见！
