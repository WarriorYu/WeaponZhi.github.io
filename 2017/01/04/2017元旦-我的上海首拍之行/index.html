2017元旦-我的上海首拍之行(第一天)

发表于 2017-01-04 | 分类于 生活 | | 本文浏览量 次

在元旦之前，我让父母把家里的550d单反寄了过来，虽然是6.7年前的入门机型，但对于我这样一个准备第一次扫街的新人小白来说却是够用了，但两天的摄影结束后，我深感我对摄影有着浓厚的兴趣，同时也切身感受到了这个机子的力不从心，准备年后换一个好镜子或者机子。

在上海的两天，老爸帮我在上海那边联系了一位叔叔，叔叔非常热情，不仅帮我订了两晚非常上档次的酒店房间，帮我了个大忙，真的非常感谢他，听闻我来摄影，还主动的借我三脚架用，可惜的是早上走的太早，叔叔送来的时候我已经出门了，最后我也没用上，因为太匆忙，连叔叔一张照片就没有留下，还是比较遗憾的，只能等下次来上海再来好好感谢叔叔了。

长乐路

12月31号，第一天，第一站，长乐路。
来上海之前，我就决定把第一站放在长乐路这，不仅是因为这里交通便利，离周围的各种有名的街道也很近，更主要的是这条街道本身也大有文章，这里有数不清的个性小店，时尚和市井气息的交融也让这条街道格外有味道

当然，随处可见的mobike和ofo当然是少不了拍


我觉得还是mobike酷点，银色更适合我，哈哈。可惜这次旅行最蠢的就是身上背的东西太多了，脖子上挂着单反，所以到最后也没试着骑一把爽爽，只能下次再来试了。

长乐路的天桥上，一位骑行者

第一天的陪伴者：雅婷，不过妹子不愿意让我给她拍，可能是嫌我刚上手单反还太菜了吧。。所以我只能偶尔偷拍下她的背影，长乐路的古朴小街上，她时尚的身段却毫不违和

外面全是时尚小店，弄堂里却生活市井气息十足

复兴中路

复兴中路这边有很多教堂建筑，还有一家特别有意思的时尚店，这里的老外很多，因为曾经是上世纪法租界领地，所以建筑风格很是别致，这个过程中路过了一个很大的公园，拍了很多有意思的照片。

勉强上个镜，哈哈。

这张照片我挺喜欢的，公园里不论老少都有很多人在玩空竹，但反正我不会玩哈哈

嗯，这个我会玩，不过也是好多年前的事了…大叔吹的不错，引起了围观

上海看到最多的就是路边高大的法式梧桐，当然还有各种名人故居

小文艺….

雅婷的头发真的很亮

活久见系列，妹纸终于露脸啦！话说这种文艺小店来拍照的人真的多！

南京路

世界文明的南京路，时尚的代名词，随便拍~

所以说，这车真的好看！

即使是一些繁华地带，上海也总是能透露出一种厚重的气质，很有味道

吃午饭啦！好像有人偷看我的镜头了？

画面上方的高楼和画面中央的老式门面形成鲜明的对比，窗户外挂的一件件衣服总有一种要把我拉回到80年代街道的错觉

盘曲而古朴的电线

南京西路中间的m豆主题店一角

抓拍

外滩

重头戏来了，因为快到晚上了，赶紧跑到外滩准备大干一场！

因为去年的某些恶劣事故，所以外滩随处可见武警

外滩边的银行群

魅力外滩



感觉有点冷啊

情深深雨蒙蒙

因为技术有限，相机能力有限，再加上没带三脚架，所以整体的广角拍的质量都不算特别好，留下了遗憾，也为下次上海之行作了铺垫，哈哈


城隍庙

这里是小吃街，同时也有很多中国风的寺庙一样的建筑，同时街边也有大量很好玩的小店，晚上非常漂亮！只可惜还是因为个人能力…….唉，将就看吧！


这家店，印象中在网上买过他家的梳子，做的还是非常精致的，当时母亲节送给妈妈的

差不多拍到这里单反就没电了，才拍了500张左右就没电了，很尴尬，可能是因为我不拍的时候没有把电源关了的缘故吧，不过后来外滩的人就多到恐怖了，所以为了避免回不去，提前坐地铁回去了，一路上除了人还是人，也没啥好拍的，街上到处都是武警，外滩周围的几站地铁都停了，实际上当时还是有点慌得，毕竟人真的多到恐怖，如果摔倒了的话后果不堪设想，所以出行还是得注意安全，也不一定非要在假日，如果在人少的时间去，会轻松很多，摄影起来也不会因为人多而很难拍。

到此第一天的上海之行结束
