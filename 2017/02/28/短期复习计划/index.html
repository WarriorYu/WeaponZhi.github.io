短期复习计划

发表于 2017-02-28 | 分类于 计划 | 本文浏览量 次

这次离职是裸辞而不是边工作边找工作的方式有几个原因：

  • 公司让我去苏州，如果先去苏州又得折腾一阵，而且不方便去南京面试
  • 年前有一次小米面试的经历，过了所有的技术面试但终面没过，让我觉得非常可惜，因为这次我是被动被通知要求面试的，之前没有任何准备，错过了这么好的机会非常可惜，所以希望能空出来一到两个月时间好好准备
  • 在无锡项目组每天的状态都不是很好，想用一段时间调整下

以后这个博客暂时只用来更新些生活上的事情，记录些想法，技术上的博客准备移驾到CSDN，因为我没有对个人博客进行处理，所以百度和谷歌暂时没法搜到这个博客，以后也只希望一些朋友和那些关注我的人看到这个博客，所以也不准备去处理搜索了，这也能够帮助我筛选一下社交圈子，要精简自己的圈子。

至于CSDN，我会先把之前大量的印象笔记投放到CSDN上去，毕竟以前只顾自己记录，现在想想应该有一定的分享精神，让所有的程序员互相参与进来，所谓的开源精神就是这样，不仅造福别人，也可以顺便在和别的高手交流过程中提高下自己。而且CSDN可以直接被搜索引擎搜索到，也可以让更多人了解到我。我有一定的虚荣心，渴望被别人认可和关注，可能这就是Coder的悲哀吧，孤单孤独，渴望认可。

CSDN地址：我的CSDN博客

目前的情况还是比较严峻的：我现在手上还有2000元，下个月20号工资因为提前扣了公积金，所以扣了很多，算了下大概到手能拿到5000，下个月信用卡、花呗、白条加上新的三个月房租大概加一起要超过4500，所以最坏情况就是我只有2000块左右的资金，这就是我在找到工作前的全部家当了，每天吃饭40块，也就是说50天左右，在50天左右，不仅要复习的让我自己满意，而且要找到工作，形势严峻，不由得自己懒散了。

所以下面列一下每日的日程和复习计划：

  • 8：00 起床洗漱，吃早饭
  • 8：30 Java学习
  • 12：00 午饭
  • 12：30 休息会
  • 14：00 Android学习
  • 17：00 健身房
  • 18：10 晚饭
  • 18：40 Android学习、代码观看、博客整理
  • 21：00 娱乐时间
  • 22：30 代码温习
  • 0：00 睡觉

因为我以前18点左右去健身房总是人太多，实际上我现在去健身房主要是跑步，做一些有氧运动，保持自己的气血流畅即可，所以17点去，锻炼一小时后去吃饭感觉比较合适，避免了时间，也可以防止像以前那样吃完饭再去健身的坏习惯，实际上每天我还是需要做一些拉伸运动的，颈椎和韧带的拉伸对健康很有益。

下面是复习计划，这里提一下，这个计划应该是要在一个月左右完成，否则春招可能也靠后了，身上的资金可能也不允许，但最终的底线是我自己对自己的技术水平满意为止，否则，一切都是空谈，如果准备好，有些机会没把握好，我不会觉得遗憾，但如果自己准备不好，因为懒散而导致失败，我不会饶恕自己。

  • Thinking in Java最后剩余的看完，并重新快速复习一遍之前记录过的要点
  • 买一本JVM书，过一遍
  • 将无锡项目组的代码不是自己写的部分比如订单和消息中心，逻辑理清一遍
  • 将现在比较火的开源代码如RxJava，OkHttp，MVP框架过一遍
  • 复习一下几个自定义View的编写以及一些动画的编写逻辑
  • 复习一下几个要点问题：内存泄漏，ANR，复杂问题的解决等较深但面试常问的问题
  • 数据结构大致过一遍，不需要把代码都写出来，但把大概思路和算法理清
  • 常规的终面问题：自我介绍，自我评价，自我价值预期，需要提出的问题等

这里比较重点和核心的是JAVA两本书和项目上代码逻辑的复习，根据小米技术面试的经验，现在比较偏向于一些基础和项目经验，还有一些平时开发过程中难以处理问题的解决。

同时希望这段时间能提升下自己的心性，客服自己的一些心理上的梦魇，通过断舍离，净化自己的思想。

不管怎么样，这个过程也是一个锻炼自己克制力的过程，如果一个人没有克制力，将不会大成，即便是像我这样曾经是一位资深电子竞技玩家，也需要通过克制力至少达到该干嘛的时候干嘛的境界。
